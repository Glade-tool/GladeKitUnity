# GladeKitUnity
Agentic AI for Unity Game Dev

## Overview

GladeKit for Unity is an agentic AI assistant that enables direct interaction with Unity through natural language. Describe what you want to build, and the AI will create scripts, manipulate GameObjects, add components, and perform other Unity operations automatically.

## Quick Start

1. **Before Starting**: Sign up for an account at GladeKit.com to get an active API Key.
2. **Open the AI Assistant**: `Window → Glade → AI Assistant`
3. **Set your API key**: Click the settings gear icon and paste your API key, hit save or validate to.
4. **Start building**: Type what you want to create and press Enter
